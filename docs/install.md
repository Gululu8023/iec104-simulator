# 安装说明

本文档说明如何下载安装和使用 IEC104 协议模拟器发布版本。

## 下载安装

1. 打开仓库的 [Releases](../releases) 页面
2. 进入最新版本
3. 根据需要下载对应安装包：
   - `iec104-simulator-master_x.x.x_x64-setup.exe`
   - `iec104-simulator-slave_x.x.x_x64-setup.exe`
4. 双击安装包，按向导完成安装

## 安装包说明

### Master 安装包

`iec104-simulator-master_x.x.x_x64-setup.exe`

适用于以下场景：

- 作为主站连接从站设备
- 发起总召、遥控、遥调、时钟同步等操作
- 查看通信报文、SOE 和文件传输记录
- 调试或验证从站实现

### Slave 安装包

`iec104-simulator-slave_x.x.x_x64-setup.exe`

适用于以下场景：

- 作为从站监听主站连接
- 响应总召、控制命令和时钟同步请求
- 导入点表并模拟设备数据
- 执行波形模拟、变化上送、场景注入等测试

## 如何选择

- 只需要模拟主站：安装 `master`
- 只需要模拟从站：安装 `slave`
- 需要本机联调：建议两个都安装

## 系统要求

当前版本支持：

- Windows 10 x64
- Windows 11 x64

如后续发布 Linux 或 macOS 版本，会在对应 Release 页面中单独说明。

## 首次运行建议

### 主站模式

首次使用建议按以下顺序操作：

1. 创建连接配置
2. 配置目标从站 IP、端口和公共地址
3. 建立连接
4. 执行总召
5. 查看数据点与报文监控结果

### 从站模式

首次使用建议按以下顺序操作：

1. 创建监听配置
2. 配置监听地址、端口和公共地址
3. 导入点表
4. 启动监听
5. 等待主站接入并观察报文

## 常见问题

### 1. Windows 提示“未知发布者”

如果安装时出现安全提示，属于未签名安装包的常见现象。  
请确认安装包来自本仓库 Release 页面后再决定是否继续安装。

### 2. 杀毒软件拦截安装包

某些安全软件可能会对桌面安装程序进行额外检查。  
建议：

- 仅从本仓库 Release 页面下载安装包
- 校验版本号和文件名
- 如有提供 `SHA256SUMS.txt`，请优先校验哈希值

### 3. 安装后无法启动

请优先检查：

- 是否为受支持的 Windows x64 系统
- 是否缺少系统运行环境
- 是否被安全软件阻止执行
- 是否安装在受限目录

### 4. 主站连接不上从站

建议检查：

- IP 和端口是否正确
- 从站是否已启动监听
- 双方公共地址配置是否一致
- 本机防火墙是否放行对应端口

### 5. 总召后没有数据

建议检查：

- 从站是否已导入点表
- 点表中的公共地址是否正确
- 从站是否成功响应总召请求
- 当前数据点是否被筛选隐藏

- 控制面板 -> 程序和功能
