# Changelog

本文档记录 IEC104 协议模拟器公开发布版本的变更历史。

格式参考 Keep a Changelog，并遵循语义化版本管理思路。  
实际版本内容以各个 Release 页面为准。

## [0.1.0] - 2026-04-13

### Added

- 发布 IEC104 协议模拟器首个公开安装版本
- 提供 `Master` 与 `Slave` 两个独立安装包
- 支持主站模式基础能力：
  - 连接从站
  - 总召
  - 遥控/遥调
  - 时钟同步
  - 文件传输
  - 通信报文监控
- 支持从站模式基础能力：
  - 监听主站连接
  - 响应总召和控制命令
  - 点表导入
  - 数据模拟与场景注入
- 支持报文解析与抓包导出
- 支持本地数据持久化

### Notes

- 当前公开发布以 Windows x64 安装包为主
- 后续版本会持续完善稳定性、易用性和文档内容
